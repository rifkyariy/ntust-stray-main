# Plan 05 — Shared UI Package

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `packages/ui` — the shared design token CSS, Lucide icon re-exports, and primitive React components (TintedPill, Eyebrow, StatusDot, Card, ProgressRow, MetricTile) used by all three Next.js apps.

**Architecture:** A simple TypeScript package with no build step — each Next.js app imports directly from `@stray/ui` source via `transpilePackages`. Design tokens live in a single CSS file copied from the design system. React components are thin, style-only primitives.

**Tech Stack:** React 18, TypeScript 5, lucide-react 0.378, CSS custom properties (no Tailwind — all inline styles per design system)

**Prerequisite:** Plan 01 complete — pnpm workspace installed.

---

```tsx
import React from 'react';

type Tint = 'green' | 'orange' | 'slate' | 'red' | 'blue';

const TINT: Record<Tint, { bg: string; color: string; border: string }> = {
  green:  { bg: '#dcfce7', color: '#16a34a', border: '#bbf7d0' },
  orange: { bg: '#fff7ed', color: '#ea580c', border: '#fed7aa' },
  slate:  { bg: '#f1f5f9', color: '#475569', border: '#e2e8f0' },
  red:    { bg: '#fef2f2', color: '#ef4444', border: '#fecaca' },
  blue:   { bg: '#eff6ff', color: '#3b82f6', border: '#bfdbfe' },
};

interface TintedPillProps {
  tint: Tint;
  children: React.ReactNode;
  style?: React.CSSProperties;
}

export function TintedPill({ tint, children, style }: TintedPillProps) {
  const t = TINT[tint];
  return (
    <span
      style={{
        display: 'inline-flex',
        alignItems: 'center',
        padding: '2px 8px',
        borderRadius: 9999,
        fontSize: 11,
        fontWeight: 700,
        fontFamily: 'var(--font-sans)',
        background: t.bg,
        color: t.color,
        border: `1px solid ${t.border}`,
        ...style,
      }}
    >
      {children}
    </span>
  );
}
```
